# Game sounds

Add these placeholder audio files (MP3 or WAV) for the voice guessing game:

- `next-card.mp3` — played when moving to the next card
- `round-complete.mp3` — played when the deck is finished
- `correct.mp3` — correct guess (optional)
- `incorrect.mp3` — incorrect guess (optional)

If files are missing, the game uses a short fallback beep via the Web Audio API.
